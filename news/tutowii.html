Comment mettre des backups obtenus de façon complètement légale sur votre Wii

Ce site ne sera jamais référencé, j'inclus donc les téléchargements ici même. (Liens indiqués comme miroir) À noter que le serveur télécharge à environ 3.2 Mb/s Date: Juin 24

Tous les programmes mentionnés (Windows x64)

  1. Télécharger des ROMs (sur romsfun, romsgames, romspedia).
  2. Décompressez le fichier. (Devrait contenir un .rvz)
  3. Ouvrez Dolphin.
  4. Supprimez le jeu précédent (conservez-le ailleurs si nécessaire).
  5. Double-cliquez sur l'écran et sélectionnez le dossier où vous avez décompressé le fichier.
  6. Sélectionnez, puis cliquez avec le bouton droit sur Convertir le fichier.
  7. Sélectionnez ISO.
  8. Convertissez et enregistrez-le dans "C://wbfs".
  9. Insérez la carte SD / le lecteur USB où vous souhaitez enregistrer le jeu.
  10. Ouvrez Wii Backup Manager.
  11. Dans l'onglet Fichier, cliquez sur Ajouter, Fichier.
  12. Sélectionnez l'ISO que vous venez de créer.
  13. Dans C://wbfs, créez un dossier nommé "NOM DU JEU [ID DU JEU]" (Tel que "Super Smash Bros. Brawl [RSBP01]").
  14. Faites glisser l'ISO dans ce dossier.
  15. Dans l'onglet Lecteur 1, sélectionnez C:
  16. Dans l'onglet Lecteur 2, sélectionnez le lecteur correct (E:, F:, etc.).
  17. Retournez à l'onglet Lecteur 1.
  18. Sélectionnez les jeux que vous souhaitez transférer sur votre lecteur (assurez-vous que la case à cocher est cochée).
  19. Cliquez sur Transférer, puis sélectionnez Lecteur 2.

    20. Cette partie est uniquement pour ajouter des couvertures et est donc optionnelle. À partir de là, vous pouvez éjecter votre support, aller dans USB loader GX, et votre jeu devrait apparaître

  20. Allez dans l'onglet Lecteur 2.
  21. Cliquez sur Couvertures, Synchroniser et Télécharger tout.
  22. Dans le dossier de Wii Backup Manager, allez dans Couvertures, puis GX.
  23. Sélectionnez les couvertures des jeux dont vous avez besoin (clic droit, copier).
  24. Ouvrez un autre onglet de l'Explorateur de fichiers.
  25. Allez dans /apps/usbloader_gx/images (Sur votre carte SD / clé USB).
  26. Collez-le ici.

    27. Note : Les mêmes étapes pour les couvertures peuvent être effectuées avec les images de disques, il suffit d'aller dans wiibackupmanager/covers/gx/disc et dans la carte SD/USB dans /apps/usbloader_gx/images/disc.

    Note : N'oubliez pas d'éjecter les lecteurs.
