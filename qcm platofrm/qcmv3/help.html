Aide - Quiz App

Fonctionnalités

Mode Édition :

  • Ajoutez des questions en utilisant le bouton "Ajouter une question".
  • Remplissez le titre, la description et l'auteur pour chaque question.
  • Sélectionnez le type de question (QCM ou Menu déroulant).
  • Remplissez les options et la réponse correcte.
  • Exportez le QCM en utilisant le bouton "Exporter QCM".

Mode Utilisation :

  • Importez un QCM en utilisant le bouton "Importer QCM".
  • Corrigez les réponses en utilisant le bouton "Corriger".
  • Revenez au Mode Édition en utilisant le bouton "Revenir au Mode Édition".

Styles

Les réponses correctes sont affichées en vert, et les réponses incorrectes sont affichées en rouge.
