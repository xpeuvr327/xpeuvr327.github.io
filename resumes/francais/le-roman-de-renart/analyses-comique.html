Même si les blagues vannes ne nous font plus rire aujourd'hui, le roman de Renart reste truffé de comiques. Voici une liste des comiques dans le roman de Renart:

Types de comiques

  • Le comique de mots

    • l'accent - peu utilisées dans cette oeuvre

    • les jurons - ex: ysengrin dans le passage du puits

    • les incorrections de language - rare

    • les jeux de mots - rare

    • le jargon - rare

    • le dialecte - parfois

  • Le comique de répétition: On fait rire le spectateur avec des répétitions (mots, gestes, situation déjà vue...) que fait le personnage.

  • Le comique de gestes

    • les grimaces - difficile à représenter

    • les déguisements - les animaux en eux-mêmes pourraient être considérés

    • les coups de bâton

    • les personnages qui se cachent

    • les chutes

  • Le comique de situation

      On fait rire le spectateur avec la situation, cocasse ou inattendue, dans lanquelle se trouvent les personnages.

      • le quiproquo (malentendu qui fait prendre quelqu'un pour quelqu'un d'autre)
      • le renversement de situation (un personnage qui était dans une situation favorable se retrouve brusquement dans une situation défavorable, ou inversement)
  • Le comique de caracètère

    On fait rire le spectateur avec les aspects excessifs, ridicules de la personnalité d'un personnage.

    • les manifestation
    • les obsessions
    • les défauts: l'avarice la prétention, lâcheté
