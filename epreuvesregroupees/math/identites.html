Mathématiques - Identités remarquables

Les identités remarquables sont des formules algébriques utiles à savoir pour résoudre des expressions rapidement.


N'OUBLIEZ PAS LE DOUBLE PRODUIT!1!!111!!!!


Voici les principales identités remarquables :

  • 1. Carré d'une somme :

    ( a + b ) 2 = a 2 + 2 a b + b 2

  • 2. Carré d'une différence :

    ( a − b ) 2 = a 2 − 2 a b + b 2 (a - b)^2 = a^2 - 2ab + b^2

  • 3. Produit d'une somme et d'une différence :

    ( a + b ) ( a − b ) = a 2 − b 2 (a + b)(a - b) = a^2 - b^2

  • 4. Quatrième identité remarquable :

    ( x + a ) ( x + b ) = x 2 + ( a + b ) x + a b (x + a)(x + b) = x^2 + (a + b)x + ab
