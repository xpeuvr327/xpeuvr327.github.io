text tools v1.1

Cette version n'incluait pas grand chose, si ce n'est que l'outil "remplacer une chaîne". Il y avait cependant un thème noir, pas encore développé dans la v2... Vous pouvez la télécharger via discord. Petite annecdote: cette version se nomme la v1.1 et non la v1 car j'ai mis un "1" en trop dans le tag "minor version" dans le créateur de l'appli.

Text Tools v2

Cette version est la première à essayer d'être clean, avec la police San Francisco (police par défaut chez apple), des boîtes d'entrée de texte centrée, et surtout, tous les outils qui fonctionnent. Cette version est téléchargeable depuis discord, encore.

Additions:

  • Outil de hachage
  • Suppression d'une chaîne dans une autre
  • Comparaison de deux chaînes de caractères
  • Longeueur d'une chaîne de caractères
  • Boîte de texte avec téléchargement (version web uniquement)
