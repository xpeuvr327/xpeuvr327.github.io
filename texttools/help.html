Comment utiliser l'outil de remplacement

1. Entrez le texte dans le premier champ de texte.
2. Entrez le terme à remplacer dans le champ "Terme à remplacer".
3. Entrez le terme de remplacement dans le champ "Remplacer par".
4. Si vous voulez que la recherche soit sensible à la casse, cochez la case "Sensible à la casse".
5. Cliquez sur "Remplacer tout" pour remplacer toutes les occurrences du terme dans le texte.
6. Cliquez sur "Remplacer suivant" pour remplacer uniquement la prochaine occurrence du terme dans le texte.
7. Cliquez sur "Définir l'entrée comme sortie" pour définir le texte modifié comme nouveau texte d'entrée et effacer les autres champs.
8. Cliquez sur "Copier dans le presse-papiers" pour copier le texte modifié dans le presse-papiers.
9. Cliquez sur "Rafraîchir" pour rafraîchir la sortie en cas de problème.

Comment utiliser l'outil de comparaison

1. Entrez le premier texte dans le premier champ de texte.
2. Entrez le deuxième texte dans le deuxième champ de texte.
3. Cochez la case "Sensible à la casse" si vous voulez que la comparaison soit sensible à la casse.
4. Cochez la case "Sensible aux accents" si vous voulez que la comparaison tienne compte des accents.
5. Cliquez sur le bouton "Comparer".
6. Le résultat de la comparaison sera affiché en dessous du bouton.
7. Si les deux textes ont la même longueur et le même contenu, le résultat sera affiché en vert.
8. Si les deux textes ont la même longueur mais un contenu différent, le résultat sera affiché en rouge.
9. Si les deux textes ont une longueur différente, le résultat sera affiché en rouge.
10. Cliquez sur "Retour à l'accueil" pour revenir à la page d'accueil.
11. Cliquez sur le lien "page d'aide associée" pour consulter la page d'aide.

Qu'est ce que l'outil de test de champs d'entrée?

Il propose plusieurs boites de texte avec différents champs, leur type est indiqué juste en haut.
Il permet de tester votre clavier en mode: texte normal, email, mot de passe, numéro de téléphone, url, recherche et nombre.

Qu'est ce que l'outil de création de mot de passe?

Il propose différentes option pour créer un mot de passe sécurisé. Il est généré localement, aucune donnée n'est collectée. Cliquez sur générer un mot de passe, copiez-le et utilisez-le.

Qu'est ce que l'outil de génération d'emoji aléatoire?

Cliquez sur emoji aléatoire pour obtenir un emoji aléatoire. Cliquez sur précédent pour revenir jusqu'à 3 emojis en arrière. Si le bouton emoji aléatoire fait apparaître une case vide, c'est probablement que votre appareil est trop ancien pour supporter l'emoji.

Comment utiliser l'outil de transformation en liste JSON

1. Entrez le texte dans la zone de texte "Entrez du texte".
2. Cliquez sur le bouton "Convertir en JSON" pour convertir le texte en format JSON.
3. Le résultat JSON sera affiché dans la zone de texte "Preview"
4. Vous pouvez copier le résultat JSON dans le presse-papiers en cliquant sur le bouton "Copier dans le presse-papiers".
